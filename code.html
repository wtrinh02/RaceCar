                        function isPalindrome(userString){
                            userString = userString.toLowerCase();
                        
                            let regex = /[^a-z0-9]/gi;
                            userString = userString.replace(regex,"");
                        
                            let revString =[];
                            let returnObj = {};
                        
                            for(let i = userString.length - 1; i>=0; i--){
                                revString += userString[i];
                            }
                        
                            if (revString == userString){
                                returnObj.msg = "Well done! You entered a palindrome!"
                            }
                            else{
                                returnObj.msg = "Sorry! You did not enter a palindrome!"
                            }
                        
                            returnObj.reversed = revString;
                        
                            return returnObj;
                        }
                        
                        function displayMessage(returnObj){
                            document.getElementById("alertHeader").innerHTML = returnObj.msg;
                            document.getElementById("msg").innerHTML=`Your phrase reversed is: ${returnObj.reversed}`;
                            document.getElementById("alert").classList.remove("invisible");
                        }

The Code is structured in three functions.

getValue

getValue is the controller function that gets the user's value and uses it to call other functions. It first makes sure that the alert pop up is inivisble whenever the app is opened by adding the bootstrap class "invisible". Then it gets the user's input using document.getElementByID and uses the resulting string to call isPalindrome to check if the string is a palindrome. Finally the function calls displayMessage to display the result.

isPalindrome

In this application, I am ignoring special characters and case when comparing for palindrome. To do so, the function takes the inputted string and changes it to all lower case using toLowerCase(). Then it uses a regular expression to check if the string contains a specific pattern. In this case the regex checks for values 0-9 and lowercase letters. Then using the replace function, the user's string gets converted to a string with no special characters. Afterwards, the function checks if the string is a palindrome by creating a reversed copy and compares it to the original string. Finally the function returns an object with properties of msg and reversed, which hold the message displayed and the reversed string compared to.

displayMessage

displayMessage is a view function that displays the alert box when the user clicks the button. It does so by replacing the alert's header and message by changing the innerHTML to the returnObj's specific properties. Then it removes the bootstrap class "invisible" to make the alert appear.
